что такое структуры данных?
В Python существуют четыре встроенных структуры данных: список, кортеж, словарь и множество.
Структура данных - может хранить какие-то данные.


зачем они нужны?
для того чтобы хранить ифнормацию и пользоваться в дальнейшем.


что такое Hash Table?
hash table - что-то вроде массива, состоящий из данных.


что такое Linked List?
linked list это тo же самое что и массив, но уже с ссылкой на другую ячейки.


в каких случаях использовать Linked List и Hash Table?
Linked List используется когда ты хочешь добавить что-то в список,но у тебя нет места
Hash Table используется когда ты не собираешься менять что-то внутри списка.
